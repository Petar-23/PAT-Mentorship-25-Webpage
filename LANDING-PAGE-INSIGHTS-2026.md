# Landing Page Insights 2026

## Aktuelle Struktur

1. **Hero Section** - Hauptüberschrift + CTA
2. **Social Proof** - Testimonials, Zahlen
3. **Features** - Was ist enthalten
4. **Pricing** - Mentorship Pakete
5. **FAQ** - Häufige Fragen
6. **Final CTA** - Conversion Push

## Bekannte Verbesserungspotenziale

### Hero Section
- [ ] CTA Button könnte prominenter sein
- [ ] Subheadline könnte kürzer/prägnanter sein
- [ ] Mehr Bewegung/Animation für Aufmerksamkeit

### Social Proof
- [ ] Testimonial-Cards könnten moderner sein
- [ ] Zahlen könnten animiert hochzählen
- [ ] Mehr Trust-Badges

### Features
- [ ] Icons könnten einheitlicher sein
- [ ] Feature-Cards hover states verbessern
- [ ] Bessere visuelle Hierarchie

### Allgemein
- [ ] Scroll-Animationen (fade-in on scroll)
- [ ] Micro-interactions bei Buttons
- [ ] Loading states verbessern
- [ ] Mobile Navigation optimieren

## Was NICHT ändern

- Die Grundstruktur der Seite
- Texte/Copy (nur Petar darf das)
- Preise
- Bilder/Videos

## Priorisierung für Nightly Jobs

**Hoch:** Kleine CSS-Fixes, Hover-States, Spacing
**Mittel:** Animationen, Transitions
**Niedrig:** Größere Component-Refactorings

## Technische Hinweise

- Framer Motion ist installiert → nutzen für Animationen
- Tailwind CSS → utility-first
- next/image für Bilder (optimiert)
- Mobile-first approach
